WEB

web site is hosted on at least one web server , which can be accessed via a network such as the Internet or a LAN . Web pages are HTML / XHTML generally written in pure string expression, but the format of the document is. Web pages are accessed via HTTP and sometimes use HTTPS encryption to provide security and privacy for people using Web page content. The web browser used by the user expresses the contents of the page as it displays the HTML markup command on the monitor. All officially accessible websites are collectively the world wide web.
